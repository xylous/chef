# chef v0.1.0

`chef` is a chess engine.

It's still in its development phase, and not usable!

## Getting started

### Reguirements

Build-time dependencies:
- (GNU) make
- a C compiler

### Installation

Clone this repository locally and build using `make`:

```
git clone https://github.com/xylous/chef chef
cd chef
make
```
